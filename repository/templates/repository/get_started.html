Getting Started

< class="docs-content"> This web-based database application manages sample and sequencing library information for the wheatear genomics project at the Vogelwarte Sempach. To start you can start retrieving sample information by searching for an individual name on the Individuals or Experiments page and click on the individual name.
Currently the database only contains experiments for the Methylomics project. This means only individuals with existing experiment data are listed on the Experiments page.
